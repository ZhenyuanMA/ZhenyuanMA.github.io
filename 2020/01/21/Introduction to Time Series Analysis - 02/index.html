First order Autoregressive process AR(1)

Classical Decomposition Model

Xt=mt+St+YtX_t = m_t + S_t + Y_tXt​=mt​+St​+Yt​

mtm_tmt​ here shows “trend”, StS_tSt​ shows seasonal patterns, and YtY_tYt​ is random “noise” component (so far we have 4 choices of noise: iid, white noise, MA(1), and AR(1))

We can remove mtm_tmt​ and StS_tSt​ to estimate YtY_tYt​, and we have two ways:

  1. estimate trend/seasonal using a “model” (filter)
  2. differencing {Xt}\{X_t\}{Xt​} to estimate trend and seasonality (filter)

Differencing（for trend)

We define the lag-1 difference as ∇Xt=Xt−Xt−1=(1−B)Xt\nabla X_t = X_t - X_{t-1} = (1-B)X_t∇Xt​=Xt​−Xt−1​=(1−B)Xt​, where BBB is known as the backwards shift operator with BXt=Xt−1BX_t = X_{t-1}BXt​=Xt−1​

We can generalize ∇\nabla∇ and BBB to general lags by taking powers:

BjXt=Bj−1(BXt)=Bj−1Xt−1=...=Xt−jB^j X_t = B^{j-1} (BX_t) = B^{j-1}X_{t-1} = ... = X_{t-j}BjXt​=Bj−1(BXt​)=Bj−1Xt−1​=...=Xt−j​

Xt−Xt−j=(1−Bj)XtX_t - X_{t-j} = (1-B^j)X_tXt​−Xt−j​=(1−Bj)Xt​

As ∇jXt=∇(∇j−1Xt)\nabla^jX_t = \nabla(\nabla^{j-1}X_t)∇jXt​=∇(∇j−1Xt​), we have:

∇2Xt=∇(∇Xt)=∇((1−B)Xt)=(1B)(1−B)Xt=(1−2B+B2)Xt=Xt−2BXt+B2Xt=Xt−2Xt−1+Xt−2=(Xt−Xt−1)−(Xt−1−Xt−2)\nabla^2X_t = \nabla(\nabla X_t) = \nabla((1-B)X_t) = (1_B)(1-B)X_t \\=(1-2B+B^2)X_t = X_t -2BX_t + B^2X_t \\=X_t-2X_{t-1}+X_{t-2}\\=(X_t-X_{t-1}) - (X_{t-1}-X_{t-2})∇2Xt​=∇(∇Xt​)=∇((1−B)Xt​)=(1B​)(1−B)Xt​=(1−2B+B2)Xt​=Xt​−2BXt​+B2Xt​=Xt​−2Xt−1​+Xt−2​=(Xt​−Xt−1​)−(Xt−1​−Xt−2​)

∇Xt=∇(mt+Yt)=∇mt+∇Yt=mt−mt−1+Yt−Yt−1=(a+bt)+(a+b(t−1))+Yt−Yt−1=b+Yt−Yt−1\nabla X_t = \nabla(m_t+Y_t) = \nabla m_t + \nabla Y_t \\=m_t-m_{t-1} + Y_t -Y_{t-1} = (a+bt) + (a+b(t-1)) + Y_t-Y_{t-1} \\=b+Y_t-Y_{t-1}∇Xt​=∇(mt​+Yt​)=∇mt​+∇Yt​=mt​−mt−1​+Yt​−Yt−1​=(a+bt)+(a+b(t−1))+Yt​−Yt−1​=b+Yt​−Yt−1​
