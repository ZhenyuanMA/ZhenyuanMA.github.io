Time series

{Xt}\{X_t\}{Xt​} is a collection of random variables where \(t\) is the index of time.

The process of dealing with time series
  1. Describe by plotting to have concise summary of data
  2. Explain by probabilistic models (joint distributions)
  3. Predict to attain more uncertainty

Note that XtX_tXt​ is mutual independent, so we have the joint distribution Pr(X1≤x1,X2≤x2,...,Xn≤xn)=∏i=1nPr(Xi≤xi)Pr(X_1 \leq x_1, X_2 \leq x_2, ..., X_n \leq x_n) = \prod_{i=1}^n Pr(X_i \leq x_i)Pr(X1​≤x1​,X2​≤x2​,...,Xn​≤xn​)=∏i=1n​Pr(Xi​≤xi​). But for most complex models we assume that Pr(X1≤x1,X2≤x2,...,Xn≤xn)=Pr(X1≤x1)Pr(X2≤x2∣X1≤x1)...Pr(Xn≤xn∣X1≤x1...Xn−1≤xn−1)Pr(X_1 \leq x_1, X_2 \leq x_2, ..., X_n \leq x_n) = \\ Pr(X_1 \leq x_1)Pr(X_2 \leq x_2 | X_1 \leq x_1) ... Pr(X_n \leq x_n|X_1 \leq x_1 ... X_{n-1} \leq x_{n-1})Pr(X1​≤x1​,X2​≤x2​,...,Xn​≤xn​)=Pr(X1​≤x1​)Pr(X2​≤x2​∣X1​≤x1​)...Pr(Xn​≤xn​∣X1​≤x1​...Xn−1​≤xn−1​).

Semi-parametric model

In semi-parametric models, we do not specify pdf and cdf of random variables, instead we specify E(Xt)E(X_t)E(Xt​) and Cov(Xt,Xt+j)Cov(X_t, X_{t+j})Cov(Xt​,Xt+j​).

Examples
  1. iid noise: let E(Xt)=0,∀tE(X_t)=0, \forall tE(Xt​)=0,∀t and Pr(X1≤x1,X2≤x2,...,Xn≤xn)=∏i=1nPr(Xi≤xi)=∏i=1nF(Xn)Pr(X_1 \leq x_1, X_2 \leq x_2, ..., X_n \leq x_n) = \prod_{i=1}^n Pr(X_i \leq x_i) = \prod_{i=1}^n F(X_n)Pr(X1​≤x1​,X2​≤x2​,...,Xn​≤xn​)=∏i=1n​Pr(Xi​≤xi​)=∏i=1n​F(Xn​) where F(⋅)F(\cdot)F(⋅) is cumulative distribution function.
  2. random walk: let {Xt}\{X_t\}{Xt​} be iid noise, and St=X1+X2+...+XtS_t = X_1 + X_2 + ... + X_tSt​=X1​+X2​+...+Xt​. Here StS_tSt​ is a random walk. (Note that here StS_tSt​ are not independent, but E(St)=0E(S_t)=0E(St​)=0)
Models with structures

Let {Yt}\{Y_t\}{Yt​} be a time series where E(Yt)=0,∀tE(Y_t)=0, \forall tE(Yt​)=0,∀t. Let Xt=mt+YtX_t=m_t + Y_tXt​=mt​+Yt​, where mtm_tmt​ is a slowly changing function of time. (Note that YtY_tYt​ here is what makes E(Xt)=0E(X_t)=0E(Xt​)=0, but mtm_tmt​ here is what makes E(Xt)≠0E(X_t) \neq 0E(Xt​)​=0)

Some choices for mtm_tmt​ including linear function of ttt and polynomial function of ttt.

Models with seasonal variation (periodicity)

Let Xt=St+YtX_t = S_t + Y_tXt​=St​+Yt​, where E(Yt)=0,∀tE(Y_t)=0, \forall tE(Yt​)=0,∀t and StS_tSt​ is a periodic function with period ddd (i.e. St−d=StS_{t-d}=S_tSt−d​=St​).

Common choices for StS_tSt​ including sum of harmonic functions St=a0+∑j=1k(ajcos⁡(λjt)+bjsin⁡(λjt))S_t = a_0 + \sum^k_{j=1}(a_j \cos(\lambda_j t) + b_j\sin(\lambda_j t))St​=a0​+∑j=1k​(aj​cos(λj​t)+bj​sin(λj​t)), where aja_jaj​ and bjb_jbj​ are estimated, λj\lambda_jλj​ are fixed frequencies.

General strategy for analysis
  1. Plot the data to
    1. identify potential signal (trend, seasonal)
    2. identify possible models for the rsifual process
    3. identify outliers and other weird things
  2. Remove the signal
  3. Choose a model to fit the resifual and esitimate the dependence
  4. Forecast by inventory projected residuals
Why we focus on the residuals (i.e. Xt−mt^,Xt−St^X_t - \hat{m_t}, X_t - \hat{S_t}Xt​−mt​^​,Xt​−St​^​)

Let Wi∼iidN(μ,σ2)W_i \overset{\text{iid}}{\sim}N(\mu, \sigma^2)Wi​∼iidN(μ,σ2), then we have Wi−μ∼N(0,σ2)W_i - \mu \sim N(0, \sigma^2)Wi​−μ∼N(0,σ2). Now we can estimate μ\muμ to remove the signal, and also we can estimate σ2\sigma^2σ2.
